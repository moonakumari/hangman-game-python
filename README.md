# 🎮 Hangman Game (Python)

This is a simple Hangman game built using Python.

## 📌 Features:
- Random word selection
- User guesses letters
- 6 wrong guesses allowed
- Win/Lose system

## 🧠 Concepts Used:
- random module
- loops (while)
- if-else conditions
- strings and lists
- input/output
